About ShopLite

ShopLite is a beginner-friendly full-stack demo store built with HTML, CSS, JavaScript, Node.js, and Express. It helps you understand product listing, product details, and cart logic without payment integration.

What this project teaches

  • How to render products dynamically with JavaScript
  • How to store cart and product data in localStorage
  • How to build simple REST APIs using Express
  • How to create responsive layouts with CSS Grid and Flexbox
